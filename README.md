# Indi Connect

Indi Connect is a church management system concept built with Next.js for Vercel deployment. It includes a ministry landing experience, a member dashboard, contribution charts, stewardship summaries, and M-Pesa payment guidance.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the app.

## Deploy to Vercel

1. Push the project to GitHub.
2. Import the repository in Vercel.
3. Vercel will detect the Next.js app automatically and deploy it.

## Project highlights

- Church-focused landing page
- Member dashboard with financial-style charts
- Stewardship summary cards
- M-Pesa payment routing UI
- API routes for dashboard summary and payment confirmation
